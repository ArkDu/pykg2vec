Supported Dataset
==================

FreebaseFB15k
--------------

Download Freebase_ dataset.

WordNet18
------------
Download WordNet18_ dataset.

WordNet18RR
-------------

Download WordNet18RR_ dataset.

-------------
Download YAGO_ Dataset.

DeepLearning50a
----------------
Download DeepLearning_ dataset.
